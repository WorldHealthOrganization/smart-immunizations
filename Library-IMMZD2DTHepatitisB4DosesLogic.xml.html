 * Library: IMMZD2DTHepatitisB4DosesLogic (IMMZ.D2.DT.Hepatitis B.4 doses)
 * Rule: Determine if the client is due for a hepatitis B vaccination according to the national immunization schedule
 * Decision Table: 4-dose schedule (birth dose + 3 primary series doses), on-time start
 * Trigger: IMMZ.D2 Determine required vaccination(s) if any
 */
library IMMZD2DTHepatitisB4DosesLogic

using FHIR version '4.0.1'
include FHIRHelpers version '4.0.1'

parameter Today Date default Today()

/*
@dynamicValue: Guidance

  case 
    when &quot;Client is not due for a hepatitis B vaccination&quot; then &quot;Client is not due for a hepatitis B vaccination Guidance&quot;
    when &quot;Client is due for a hepatitis B vaccination&quot; then &quot;Client is due for a hepatitis B vaccination Guidance&quot;
    when &quot;Hepatitis B immunization schedule is complete&quot; then &quot;Hepatitis B immunization schedule is complete Guidance&quot; 
    else ''
  end
  
define &quot;Has Guidance&quot;:
  &quot;Guidance&quot; is not null and &quot;Guidance&quot; != ''

@output: Client is not due for a hepatitis B vaccination Case 1
@pseudocode: &quot;Immunization recommendation status&quot; = &quot;Not due&quot;
*/

  Encounter.&quot;No hepatitis B primary series doses were administered&quot;

  Encounter.&quot;Two hepatitis B primary series doses were administered&quot;
    and Encounter.&quot;The latest hepatitis B dose was administered less than 4 weeks ago&quot;

/*
@output: Client is not due for a hepatitis B vaccination
@pseudocode: &quot;Immunization recommendation status&quot; = &quot;Not due&quot;
*/
define &quot;Client is not due for a hepatitis B vaccination&quot;:
  &quot;Client is not due for a hepatitis B vaccination Case 1&quot;
    or &quot;Client is not due for a hepatitis B vaccination Case 2&quot;
    or &quot;Client is not due for a hepatitis B vaccination Case 3&quot;

/*
@output: Client is not due for a hepatitis B vaccination Guidance
@guidance = Should not vaccinate client with first hepatitis B dose as client's age is not within appropriate age range.

@guidance = Should not vaccinate client with second hepatitis B dose as the latest hepatitis B dose was administered less than 4 weeks ago.

Check for any other vaccines due, and inform the caregiver of when to come back for the third dose.
*/

  case

Check for any other vaccines due and inform the caregiver of when to come back for the first dose.'

Check for any other vaccines due and inform the caregiver of when to come back for the second dose.'
    when &quot;Client is not due for a hepatitis B vaccination Case 3&quot; then 'Should not vaccinate client with third hepatitis B dose as the latest Hepatitis B dose was administered less than 4 weeks ago.
Check for any other vaccines due, and inform the caregiver of when to come back for the third dose.'
    else ''
  end

@output: Client is due for a hepatitis B vaccination Case 1

*/

@output: Client is due for a hepatitis B vaccination Case 2

*/

    or &quot;Client is due for a hepatitis B vaccination Case 3&quot;

Check for contraindications.

Check for contraindications.

Check for contraindications.'
    when &quot;Client is due for a hepatitis B vaccination Case 3&quot; then 'Should vaccinate client with third hepatitis B dose as the latest Hepatitis B dose was administered more than 4 weeks ago.
Check for contraindications.'
    else ''
  end
